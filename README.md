# Dyar-Salih Projects

A collection of web applications and projects.

## Projects

### 1. Random Joke Generator
A web application that fetches and displays random jokes from an external API.

- **Features:**
  - Fetch jokes from JokeAPI
  - Display jokes in real-time
  - Support for different joke types
  - Responsive design
  - Dark/Light mode toggle
